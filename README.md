# Gerenciamento-Estetica
Desenvolvimento de um software para o gerenciamento de pequenos comércios, com foco em salões de beleza e barbearias. O sistema busca facilitar a organização dos atendimentos por meio do agendamento de horários, permitindo o controle de serviços, funcionários e horários disponíveis, evitando conflitos e melhorando o atendimento ao cliente.
